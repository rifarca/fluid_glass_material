# Fluid Glass Material

iOS-like fluid glass effect for Flutter with automatic performance adaptation.

## ✨ Features

- Blur glass (BackdropFilter)
- Smart auto performance mode
- Optimized for lists (no lag)
- Noise (grain) support
- Smooth animations
- Simple API

## 🚀 Installation

```yaml
dependencies:
  fluid_glass_material: ^0.0.3
